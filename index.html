Keyboard Shortcuts

Macro Editor Shortcuts

`Meta + S` / `Meta + Enter`Save current macro to device and restart runtime
`Meta + E`Export current macro to file
`Meta + O`Import from file
`Meta + Shift + N`Create new macro
`Meta + Shift + H`Toggle help
`Meta + Shift + G`Toggle log
`Meta + Shift + M`Toggle macro sidebar
`Ctrl + Space`Auto-complete in Monaco
`Meta + Z` / `Meta + Shift + Z`Undo / Redo in Monaco
`Meta + F`Search in Monaco
`Meta + /`Toggle comment in Monaco
`Tab` / `Shift + Tab`Indent / un-indent in Monaco
